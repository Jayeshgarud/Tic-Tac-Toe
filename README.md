# Tic Tac Toe Game (C++)

A simple console-based Tic Tac Toe game developed in C++. This project demonstrates fundamental programming concepts such as arrays, loops, functions, and conditional statements through an interactive two-player game.

## Features

- Two-player gameplay (Player X and Player O)
- Dynamic game board updates after every move
- Win detection
- Draw detection
- Input validation
- Replay option
- Simple console interface

## Concepts Used

- Arrays
- Loops
- Conditional Statements
- Functions
- User Input/Output
- Basic Game Logic

## Gameplay

1. The game displays a board with positions numbered 1–9.
2. Player X starts first.
3. Players take turns selecting an available position.
4. The board updates after each move.
5. The game ends when:
   - A player gets three symbols in a row, column, or diagonal.
   - All positions are filled, resulting in a draw.
6. Players can choose to play another round.
